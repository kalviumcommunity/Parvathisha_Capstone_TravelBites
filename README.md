# 🧳 Travel Bites: An Integrated Travel and Restaurant Services Web Application

## 📝 Brief Description

**Travel Bites** is a frontend-only web application built using **React JS**, **Tailwind CSS**, and **ShadCN UI**, designed to simplify travel planning and restaurant reservations — especially for **pilgrims**.  
This 3-week capstone project focuses on creating a **responsive, interactive** UI using **mock APIs** and static data to simulate bookings, restaurant lists, and itinerary plans.

---

## 🚀 Tech Stack

- ⚛️ React JS (with Vite)
- 🎨 Tailwind CSS
- 🧩 ShadCN UI
- 🔥 Firebase (mocked)
- 🌐 Mock APIs / Static JSON
- 📍 Map APIs (mocked integration)
