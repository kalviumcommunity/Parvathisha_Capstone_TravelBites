Travel Bites: An Integrated Travel and Restaurant Services Web Application

Brief Description:
Travel Bites is a frontend web application built using React JS, Tailwind CSS, and ShadCN UI, designed to simplify travel planning and restaurant reservations—especially for pilgrims. This 3-week project focuses solely on building a fully functional, responsive frontend with mock APIs and static data integration to simulate booking experiences, interactive UI, and dynamic components.
